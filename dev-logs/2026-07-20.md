### ❌ 遇到的问题
1. **Electron 下载超时**：npm 从 GitHub 下载二进制文件被墙，镜像站的文件又损坏
   - 解决：切换到 Python + rumps + PyObjC 方案
2. **PyObjC 12.0 编译超时**：最新版需要本地编译 C 扩展
   - 解决：降级到 PyObjC 10.3.2（有预编译 wheel）
3. **WKScriptMessageHandler 导入失败**：它是 ObjC 协议不是 Python 类
   - 解决：移除显式导入，PyObjC 通过方法签名自动匹配
4. **NSWindowStyleMask 枚举不可用**：PyObjC 中此类型不直接暴露
   - 解决：使用整数值 (1|2|4|8) 替代
